Questão 2: Arquitetura Cloud AWS

Componentes da Solução

Frontend

S3 + CloudFront

Hospedagem dos arquivos estáticos React

Distribuição global via CDN para baixa latência e alta disponibilidade

Banco de Dados

AWS RDS

Banco relacional gerenciado (Postgres/MySQL)

Backups automáticos e escalabilidade vertical/horizontal

ElastiCache (Redis)

Cache para acelerar consultas frequentes

Reduz carga no banco e melhora tempo de resposta

Backend

Docker + ECS Fargate

Containers Node.js orquestrados e autoescaláveis

Execução serverless com capacidade sob demanda

Application Load Balancer

Distribuição de tráfego entre containers

Health checks e roteamento inteligente

Segurança e DevOps

IAM + Security Groups + WAF

Controle de acesso e proteção contra ataques

CloudWatch + CodePipeline

Monitoramento e automação de deploy CI/CD

Escalabilidade
Alta Disponibilidade
